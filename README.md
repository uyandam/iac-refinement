# Masters Project

An agentic system that uses Claude AI to provision and manage a local Kubernetes VM via SSH and Ansible.

---

## What It Does

- Connects to a local VM over SSH
- Writes and uploads Ansible playbooks dynamically
- Uses Claude as an AI agent to decide and execute deployment steps
